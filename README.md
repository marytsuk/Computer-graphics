# Computer-graphics
OpenGL and GLSL

Implemented effects:
  1. The standard lighting model (Phong);
  2. Layered fog;
  3. Normal Mapping (spider);
  4. Non-standard lighting model (Cook-Torrens, ball);
  5. Fuzzy shadows (shadow map).
 Moving around the scene : A, D, S, W;
 Enable fog : F;
 Disable fog : G;
 To close the window : ESC.
